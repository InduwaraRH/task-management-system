# Client - Task Management System

Frontend built with Vite + React + TypeScript.

## Setup

- Copy `.env` values into `.env.local` if needed. Ensure `VITE_API_URL` points to backend (e.g., `http://localhost:4000`).

## Scripts

- `npm run dev` - run dev server
- `npm run build` - build for production
- `npm run preview` - preview production build

## Notes

This client is a scaffold demonstrating routing, query setup, and component structure. Forms, validation, and UI components have TODO markers for implementation.
